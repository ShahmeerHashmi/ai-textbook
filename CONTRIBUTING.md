# Contributing to Physical AI & Humanoid Robotics Textbook Generator

We welcome contributions to improve the textbook generation system. This document outlines the process for contributing to the project.

## Getting Started

1. Fork the repository
2. Create a feature branch for your changes
3. Make your changes following the code style
4. Add tests if applicable
5. Submit a pull request

## Development Setup

```bash
# Clone your fork
git clone https://github.com/your-username/project-name.git
cd project-name

# Install dependencies
npm install

# Build the project
npm run build
```

## Code Style

- Use TypeScript 5.x with strict mode enabled
- Follow the existing code structure and naming conventions
- Write clear, descriptive variable and function names
- Add JSDoc comments for public functions and classes
- Keep functions focused and small when possible

## Testing

- Write unit tests for new functionality
- Run tests before submitting changes: `npm test`
- Ensure all tests pass before submitting a pull request

## Commit Messages

- Use clear, descriptive commit messages
- Follow conventional commit format when possible
- Reference issue numbers if applicable

## Pull Request Process

1. Ensure your changes address the issue or feature request
2. Update documentation if necessary
3. Add tests for new functionality
4. Verify all tests pass
5. Submit the pull request with a clear description

## Questions?

If you have questions about contributing, feel free to open an issue for discussion.
