# ROS 2 Project

## Description

Implement a complete ROS 2 project with nodes, topics, and services. This project will demonstrate your understanding of ROS 2 architecture and communication patterns.

## Requirements

- Create at least 3 interconnected nodes
- Implement publisher/subscriber communication
- Implement service/client communication
- Use parameter servers for configuration
- Include proper error handling and logging

## Learning Objectives

- Design a multi-node ROS 2 system
- Implement various communication patterns
- Handle configuration through parameters
- Apply best practices for ROS 2 development

## Grading Criteria

- Functionality: 40%
- Code quality: 25%
- Documentation: 20%
- Presentation: 15%

## Duration

2-3 weeks

## Project Steps

1. Design the system architecture
2. Implement individual nodes
3. Set up communication patterns
4. Test and debug the system
5. Document the implementation
6. Prepare presentation materials

## Submission Guidelines

- Submit source code with README
- Include launch files for easy execution
- Provide documentation of design decisions
- Record a demonstration video
